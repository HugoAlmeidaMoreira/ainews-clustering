# Motivation

Two major factors impact solutions: **distance** + **approach**

- **distance metrics** depend on the:
- variable domains
- numeric and ordinal (e.g. Euclidean)
- nominal (e.g. Hamming)
- non-iid attributes
- data structure: tabular, time series, image, spatiotemporal data, events...
- approach
- partitioning
- hierarchical
- density-based
- model-based
