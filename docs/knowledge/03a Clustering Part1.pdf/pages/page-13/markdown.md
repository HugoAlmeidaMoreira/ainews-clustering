# Distances and metrics

A distance function is a metric if the following conditions are met:

- non-negative
$$
d(x, y) \geq 0
$$

- distance to point itself is zero
$$
d(x, x) = 0
$$

- symmetry
$$
d(x, y) = d(y, x)
$$

- triangular inequality
$$
d(x, y) \leq d(x, z) + d(z, y)
$$
