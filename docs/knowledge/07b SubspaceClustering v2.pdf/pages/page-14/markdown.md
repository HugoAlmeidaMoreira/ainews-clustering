- overall (low-variance)
$$
a_{ij} = c + \eta_{ij}
$$
- on variables
$$
a_{ij} = c_j + \eta_{ij}
$$
where...
- $c_j$ is a constant
- $\eta_{ij}$ the noise factor
- $a_{IJ}$ average $(I,J)$
- $a_{IJ}$ average $(I,\{yj\})$
