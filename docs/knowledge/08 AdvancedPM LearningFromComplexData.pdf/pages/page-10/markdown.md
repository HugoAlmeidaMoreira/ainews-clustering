# Event data

Event: timestamped occurrence
- non-variate, univariate (typed event), multivariate (e.g. high and low blood pressure)

Event sequence: set of events

Event data: set of observations, each being an event sequence

Examples:
- health records
- social interactions
- machine logs
- musical melodies
- headline events
